# StudioHitboxSet


The StudioHitboxSet object contains information about a studio models hitboxes.


## Methods


### GetName()


Returns the name of the hitbox set.


### GetHitboxes()


Returns a table of [StudioBBox](../StudioBBox/) objects for the hitbox set.
